STOA - the cleanest jQuery photos gallery out there

Features

  • vertical thumbnails layout(could't find one...)
  • LTR/RTL are supported
  • thumbnails mouse scroll support, allowing elegant layout for large amount of photos
  • browser/mobile support
  • authomatic build without any html coding necessary(for the lazy ones of us)

How to use - Api

set reference to the following files: 
  <script type="text/javascript" src="/plugins/Gallery/jquery-1.5.1.min.js"></script>
  <script type="text/javascript" src="/plugins/Gallery/jquery.ui.widget.min.js"></script>
  <script type="text/javascript" src="/plugins/Gallery/Gallery.plugin.js"></script>
  <link rel='Stylesheet' type='text/css' href='/plugins/Gallery/Gallery.css' />

  <-- optional, jquery.mousewheel.min.js for mouse wheel scroll through thumbnails support -->
  <script type="text/javascript" src="/plugins/Gallery/jquery.mousewheel.min.js"></script>

invoke stoa on the relevant element: 
  <script type="text/javascript">
    $(document).ready(function () {
      $('div#turn-me-into-stoa').stoa();
    });
  </script>
